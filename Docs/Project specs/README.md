## 🔑 Key Concepts

### 1. Cognitive Digital Twin
Nexus creates an AI replica that can:
- Act as your representative in communications
- Manage relationships with all contact types
- Execute autonomous tasks
- Continuously learn and evolve

### 2. Eight Core Hubs
| Hub | Purpose |
|-----|---------|
| **AgentsHub** | Manage AI agents and their capabilities |
| **WorkflowsHub** | Orchestrate complex multi-step operations |
| **SettingsHub** | Centralized configuration management |
| **ContactsHub** | 360-degree view of every person |
| **LogsHub** | Comprehensive audit trail |
| **MemoryHub** | Advanced cognitive memory management |
| **NexusHub** | Main dashboard and interface |
| **AIModelsHub** | Multi-provider AI orchestration |

### 3. Five-Layer Memory
- **Working Memory**: Real-time context (Redis)
- **Episodic Memory**: Events and conversations
- **Semantic Memory**: Facts and knowledge (Vector DB)
- **Structured Memory**: Database entities
- **Graph Memory**: Relationship networks

## 🛠️ Technology Stack

### Backend
- **Laravel 11.x** - PHP Framework
- **MySQL 8.0+** - Primary Database
- **Redis 7.0+** - Caching & Queues
- **Laravel Horizon** - Queue Management

### Frontend
- **Vue.js 3** - JavaScript Framework
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Alpine.js** - Reactive Components

### AI & Memory
- **Google Gemini** - Primary AI Model
- **OpenAI** - Alternative Provider
- **Anthropic** - Complex Reasoning
- **Pinecone** - Vector Database

### External Integrations
- **WAHA** - WhatsApp HTTP API
